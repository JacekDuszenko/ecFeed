-------------------------------------------------------------------------------------------
       Installation and running of ecFeed For Mac OS (standalone application)
-------------------------------------------------------------------------------------------

- Install latest version of Java 
- Unzip ecfeed_osx_X.Y.Z.zip to any empty directory.
- In the directory where you unzipped the file, go to subdirectory: MacOS
- Disable MacOs Gatekeeper program: Settings > Security & Privacy > General > Allow aps downloaded from: Anywhere
- Double click the 'eclipse' executable file. After a while the main window of ecFeed will be displayed.
- Enable MacOs Gatekeeper
