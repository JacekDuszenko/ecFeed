
-------------------------------------------------------------------------------------------


       Installation and running of ecFeed For Windows (standalone application)

-------------------------------------------------------------------------------------------



- Install latest version of Java 

- Unzip ecfeed_win64_X.Y.Z.zip to any empty directory.

- In the directory where you unzipped the file, go to subdirectory: eclipse


- Double click the eclipse.exe file. After a while the main window of ecFeed will be displayed.

- For convinience there is a batch file ecfeed.bat provided in the top folder which does two latest steps for you.
