-------------------------------------------------------------------------------------------
       Installation and running of ecFeed For Linux (standalone application)
-------------------------------------------------------------------------------------------

- Install latest version of Java 
- Unzip ecfeed_win64_X.Y.Z.zip to any empty directory.
- In the directory where you unzipped the file, go to subdirectory: eclipse
- Double click the 'eclipse' executable file. After a while the main window of ecFeed will be displayed.
- For convinience there is shell file ecfeed.sh provided in the top folder which does two latest steps for you.
